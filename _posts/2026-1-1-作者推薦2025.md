又來到一年的最後一天，鄭重地向各位陪伴許久的讀者們問好，今年將要邁向第五個年頭了。

大家今年都過得好嗎？希望大家都有像作者去年結尾許的願那般成為自己想成為的人，至少不違背自己的想法，快樂地做出大部分的選擇。

那麼就照慣例來回顧今年寫的作品吧！

1. [【紗瑜】如果明天會到來](https://thekingwithdonkeyears.com/category/%E5%A6%82%E6%9E%9C%E6%98%8E%E5%A4%A9%E9%82%84%E6%9C%83%E5%88%B0%E4%BE%86%E7%9A%84%E8%A9%B1/)

```
她們曾經失去過一些東西，以為人生會就此停在那刻，卻因為彼此，讓停止轉動的齒輪又重新運作。

她們錯過太多，已經不是一個擁抱能解決的問題。

她們需要明天，很多很多的明天，去把握過去錯過的，去實現未曾實現的，去期待還未經歷過的。

如果還有明天，那一定會比今天更幸福，那也一定能開始期待永遠的幸福快樂吧。
```

上面這段就是作者一開始想像好的結尾。

作者也在結語的地方提到過，這篇是作者在進擊的巨人完結後寫下的草稿，放了很久才又拿出來，想寫的是一個大戰過後歸於平靜的感覺。

壯烈的犧牲後，如果還有明天，就能拿來談情說愛吧。

2. [【Saida/定Mo/玉米】氣象觀察站](https://thekingwithdonkeyears.com/category/%E6%B0%A3%E8%B1%A1%E8%A7%80%E5%AF%9F%E7%AB%99/)

```
雨季過後，彩虹帶來什麼？
```

故事主軸依然圍繞著凑崎和金多賢以及他們生命中的彩虹，這次加上真的來自宇宙的外星人與活了上百年的雪女，還有一對看得見鬼的夫妻檔，三組人聯手帶迷路小孩回家。

在寫下這行字時氣象觀察站還沒完結，但作者大膽的把最後一篇的日期壓在31號，不知道是否能達成？

要說作者從寫這篇文學到的，就是上班族寫文不要把目標放在一週出一萬字，只要有一千字就已經很棒。如果扛著一萬字的壓力，下班後寫文會非常痛苦，假日也會覺得自己在交差了事。

不過也深刻感受到一週至少要六千字的鋪陳才可以把故事說的完整，這篇寫了三十篇，但全文大概只有四萬字左右，撐不起原本的大框架。基於不想把故事拖得太長，最後只能刪減一些內容。

在下面告訴作者你的遺憾，我們來看看有沒有辦法解決？

3. [【satzu】 幸福嗎？](https://thekingwithdonkeyears.com/alphaxomega-%E5%B9%B8%E7%A6%8F%E5%97%8E/)

```
關於愛的定義有很多種，希望你過得幸福是其中之一。
```

作為 [【satzu】「老大。」](https://thekingwithdonkeyears.com/2022/07/26/%E3%80%90satzu%E3%80%91-%E3%80%8C%E8%80%81%E5%A4%A7%E3%80%82%E3%80%8D/) 和 [【satzu】聖誕老公公](https://thekingwithdonkeyears.com/2022/12/23/%E3%80%90satzu%E3%80%91-%E8%81%96%E8%AA%95%E8%80%81%E5%85%AC%E5%85%AC/)的續篇，故事發生在交往兩年後，凑崎因為母親的緣故宣布引退！？

和家人的相處是一輩子的課題，但只要兩人（或是三人？）一直在一起便可以有辦法解決。

他愛他，無庸置疑。 

這篇是作者在看完TWICE在台灣的第一次演唱會後寫下的小短篇，雖然沒能有幸在現場見證，但滑到影片的當下真的很感動。

原來的存稿躺在作者的資料夾已經兩年，本來的名字是取叫Never Grow Up。源自於作者一段很幸福的時光，因為太幸福，那是作者第一次希望時間過的慢一點，最好就暫停在那一刻。

當然人生都還是會繼續前進。

4. [【YouNameIt】New Rules](https://thekingwithdonkeyears.com/younameit-newrules/)

 ```
 分手後最好立下規矩，否則一不小心又會變回砲友 
 ```

簡單來說就是一個打炮的故事，至於為什麼是自己決定主角名字的形式呢？這還得回到網站的託管方式說起。

可以想成因為懶得管，以前作者都是放在專業人士那裡。不過這幾年突然意識到作者是相關科系畢業，自己管理應該是一塊小蛋糕才對，目前改為自己管理後成功省下一大筆錢，可喜可賀。也因為自己管理，作者開始開發一些奇特的功能，例如自行替換主角名稱的故事。

於是就誕生出這篇文章。

雖然是可以自己替換名字的故事，作者也不想影響大家，但作者自己在p1放的是凑崎。

以上。

至於明年會寫什麼故事呢？這還是作者第一次什麼規劃都沒有，但大概會先休息幾個禮拜。

目前想到的是要把[在結婚前要談多少次戀愛呢？](https://thekingwithdonkeyears.com/category/%E5%9C%A8%E7%B5%90%E5%A9%9A%E5%89%8D%E8%A6%81%E8%AB%87%E5%A4%9A%E5%B0%91%E6%AC%A1%E6%88%80%E6%84%9B%E5%91%A2/) 的續篇寫出來，一樣是躺在草稿兩年的東西；或是用作者很喜歡的一個美劇摩登家庭來改編寫一些輕鬆的小品，這次真的是喜劇；也有看到讀者想看米彩，但彩現在有穩交男朋友，非常可能只會是BE故事？

那又來到一點作者的近況分享。

回想今年最感動的時刻，那一定是親自參加孩子們在台灣的第一場演唱會。到現場之前，本來想著已經看過這次巡演的仁川場直播，過去也在國外看過現場，應該不會差太多。

實際體會後發現能夠在自己家鄉參加，確實是非常不一樣的體驗。

作者本來也覺得喜歡的人說自己的語言不是一件多大的事，反正英文、日文和韓文也都還算熟練，可是聽到日不落的那刻，真的暈得咪咪冒冒的。

都說高雄場的出口在台北場，作者有幸又能參加，非常快樂。

今年底作者有幸獲得升遷，明年又要搬到其他國家，預計會是更加忙碌的一年。雖然工作繁忙，但寫文果然還是作者的療癒，明年也會緩慢的寫出一些小東西。

老樣子，再次感謝各位讀者一年的支持，希望各位讀者們在2026年都成為不讓自己失望的人，別忘了常回來光臨小破站。

日常祝福親愛的孩子們與讀者們平安健康，且務必要快樂，都能夠比2025年更加幸福。
